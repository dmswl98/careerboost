## 실행 방법

1. 상위에 `.env` 파일을 생성한 뒤 [OpenAI](https://openai.com/)에서 발급 받은 키 값을 `OPENAI_API_KEY`에 할당해주세요.
2. `pnpm install`
3. `pnpm dev`
